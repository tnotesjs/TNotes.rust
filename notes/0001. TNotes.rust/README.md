## 2. 🫧 评价

该知识库主要用于记录和 Rust 学习相关的一些笔记。

在正式开始之前，简单搜了搜相关学习资料，发现官方就有推出对应的开源书籍，正好基于中文版来初始化该知识库。

## 3. 🤔 知识库创建背景是？

创建日期：`2026.06.05`

后续负责的桌面端项目计划考虑使用 Tauri 2.0 来实现。Electron 的后端是 Node.js，Tauri 的后端是 Rust。

虽然 Node.js 确实更加熟悉，但是 Electron 好像确实比较重，Tauri 2.0 打包结果更小一些，计划先学点儿 Rust 基础，后续尝试上 Tauri 2.0 试试水。
